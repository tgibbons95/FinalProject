Connect 4 Program-How to Use

compile with "g++ Connect4.cpp"

You will be given option of 1 to 2 players

Checks for four in a row across, down, and diagonally.
Checks if whole board is full. Based on if each column is full.
It ends the game when one of the prior statements is true.

Game itself is very solid but have not incorporated all of the features yet.
